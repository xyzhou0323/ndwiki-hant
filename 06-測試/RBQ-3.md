# RBQ-3 重複行爲

**RBQ-3**（Repetitive Behaviour Questionnaire 3）是一份評估重複行爲模式的工具，由 Jones 等人於2024年開發。覆蓋從15月齡嬰幼兒到成人的全年齡段，可自評也可由他人評估。

> **本站測試**：[RBQ-3 重複行爲](https://neuroxyz.cn/tests/rbq3.html)
> **中文版提示**：本測試目前僅有英文原版常模，**尚無中文版常模**。本站提供的百分位、閾值等解讀數據均基於英文原版，缺少本土化驗證。中文語境下的分數解讀存在文化差異風險，結果**僅供參考**，不構成任何臨牀判斷依據。

## 測試結構

- **20題**，約5分鐘完成
- **自評適用**：13 歲以上無智力障礙個體；他評適用於 15 月齡以上全年齡段
- **計分規則**（關鍵）：選項索引需轉換爲得分（並非簡單的 1/2/3/4）
  - 4 點量表題（Q1-Q19）：選項映射爲 **[1, 2, 3, 3]**（即第2和第3個選項同爲 3 分）
  - 3 點量表題（Q20）：選項映射爲 **[1, 2, 3]**
- **兩類量表**：
  - **Q1-Q6 頻率量表**：以每日行爲次數度量（如"從不或很少"到"每天30次或更多"）
  - **Q7-Q20 嚴重程度量表**：以行爲強度/影響程度度量（如"從不或很少"到"嚴重或頻繁"）
- 兩個子維度：
  - **重複感覺運動行爲**（RSMB）：如擺動身體、旋轉物體等
  - **堅持同一性**（IS）：對日常程序和一致性的強烈需求
- **不參與維度計分的題目**（維度標記爲"NONE"）：Q1, Q11, Q20
  - 這三題僅計入總分和總均分，不計入 RSMB 或 IS 子維度均分

## 結果解讀

### 均分分級體系

RBQ-3 使用各維度均分（而非粗分）進行分級，四檔評估：

| 等級 | 總均分 | RSMB 均分 | IS 均分 |
|------|--------|-----------|---------|
| 從不或很少 | < 1.75 | < 1.75 | < 1.78 |
| 輕微或偶爾 | 1.75 - 2.49 | 1.75 - 2.49 | 1.78 - 2.55 |
| 明顯或經常 | 2.50 - 3.24 | 2.50 - 3.24 | 2.56 - 3.32 |
| 嚴重或頻繁 | ≥ 3.25 | ≥ 3.25 | ≥ 3.33 |

### 常模對比

| 人羣 | 總均分 | RSMB 均分 | IS 均分 |
|------|--------|-----------|---------|
| 典型人羣 | ~1.35 | ~1.29 | ~1.36 |
| 孤獨譜系羣體 | ~2.48 | ~2.25 | ~2.28 |

### 分級解讀

各維度均分對應的特徵解讀：

#### 總均分

| 均分 | 解讀 |
|------|------|
| 從不或很少（< 1.75） | 行爲模式非常靈活，很少表現出重複行爲特徵，日常變化不會造成困擾 |
| 輕微或偶爾（1.75 - 2.49） | 偶爾表現出一些重複性行爲特徵，但整體靈活度良好 |
| 明顯或經常（2.50 - 3.24） | 經常表現出重複行爲特徵，可能對日常生活造成一些困擾 |
| 嚴重或頻繁（≥ 3.25） | 表現出強烈的重複行爲特徵並極度偏好穩定與一致性，可能顯著影響日常生活，對變化容易感到焦慮 |

#### RSMB（重複感覺運動行爲）

| 均分 | 解讀 |
|------|------|
| 從不或很少（< 1.75） | 少有重複的感覺運動行爲，身體動作模式靈活多樣 |
| 輕微或偶爾（1.75 - 2.49） | 偶爾表現出重複性感覺運動行爲，但整體身體動作較靈活 |
| 明顯或經常（2.50 - 3.24） | 經常表現出重複性感覺運動行爲，可能是感官自我調節策略或習慣性行爲模式 |
| 嚴重或頻繁（≥ 3.25） | 頻繁表現出重複性感覺運動行爲，可能對日常活動產生明顯影響 |

#### IS（堅持同一性）

| 均分 | 解讀 |
|------|------|
| 從不或很少（< 1.78） | 對日常變化的接受度很高，適應新環境的能力強 |
| 輕微或偶爾（1.78 - 2.55） | 對環境穩定性有一定需求，但通常能夠適應日常變化 |
| 明顯或經常（2.56 - 3.32） | 較爲依賴環境的穩定性和日常的可預測性，改變慣例可能引起不適或焦慮 |
| 嚴重或頻繁（≥ 3.33） | 高度依賴環境的穩定性和可預測性，改變慣例通常引起明顯不適或焦慮 |

原始總分範圍爲 20-79 分。RSMB 子量表原始分範圍 8-32 分，IS 子量表原始分範圍 9-36 分。結果以雷達圖可視化三個維度的均分剖面。

## 在神經多樣性範式中的位置

重複行爲在傳統[[病理學範式]]下被定義爲"刻板行爲障礙"。在[[神經多樣性範式]]下，RBQ-3 評估的這些行爲被重新理解爲**自我調節**方式——包括感官調節、情緒調節和認知調節。堅持同一性也被理解爲對可預測性的合理需求，而非僵化。

## 與其他版本的關係

RBQ-3 是對[[RBQ-2A]]（成人版）的更新和擴展，統一了全年齡段的評估標準。三個版本的演進反映了對重複行爲理解的深化：從單一維度到多維結構。
