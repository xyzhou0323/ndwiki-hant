# AASPIRE 孤獨譜系耗竭量表修訂版 (AABS-R14)

**AABS-R14**（AASPIRE Autistic Burnout Scale – Revised，14 項）是由 [[AASPIRE]] 成果項目團隊採用社區參與式研究（CBPR）方法開發的標準化量表，用於識別和測量慢性[[孤獨譜系耗竭]]。該量表有直接報告版本（自評）和照護者報告版本（他評）兩種形式，適用於伴有或不伴有智力障礙的孤獨譜系成年人，也可用於其他需要認知無障礙調查工具的人羣。

> **本站測試**：[AABS-R14 中文免費測試](https://neuroxyz.cn/tests/aabsr14.html)
> **版權聲明**：[AASPIRE](https://aaspire.org/) 孤獨譜系耗竭量表修訂版可在非商業、臨牀和研究場合免費使用，使用時需註明出處。修改、改編或盈利性使用需經 AASPIRE 授權。許可諮詢請聯繫 [[Christina-Nicolaidis]] 和 AASPIRE：info@aaspire.org。本站已取得 AASPIRE 授權翻譯和發佈完整量表內容。

## 開發背景

[[AASPIRE]] 孤獨譜系耗竭項目團隊於 2019 年創建了原始版孤獨譜系耗竭量表，條目基於關於孤獨譜系耗竭的定性研究結果（[[Raymaker2020|Raymaker et al., 2020]]）。原始版共 27 題，條目圍繞定性研究中識別的六類特徵構建（[[RaymakerEtAl2026|Raymaker et al., 2026]]）：

| 特徵類別 | 涵蓋內容 |
|----------|----------|
| 認知能力下降 | 思維/處理、決策、解決問題、記憶 |
| 情緒調節下降 | 衝動控制、情緒波動、易怒 |
| 感官敏感性增高 | 感官不耐受、過度刺激、崩潰/關閉、過濾困難 |
| 功能能力下降 | 社交/溝通、獨立生活、職業/學業、排序與計劃 |
| 迴避與退縮增加 | 社交回避、環境迴避、活動迴避 |
| 耗竭增加 | 心理耗竭、身體耗竭 |

當前推薦的修訂版爲納入 AASPIRE 成果項目（Outcomes Project）而修訂。成果項目團隊採用 CBPR 方法，旨在：1) 創建並驗證 AASPIRE Measurement Toolkit；2) 瞭解哪些因素能預測結局隨時間的改變（Nicolaidis et al., 2026a）。

修訂內容包括：
- 縮短工具長度
- 添加文字以明確時間框架
- 進行小幅修改以提高可及性
- 添加詳細描述和情景描述以更清晰定義概念
- 添加關於症狀初始發生的條目

AASPIRE Measurement Toolkit 中的每個工具均有自評和照護者報告版本。社區-學術團隊合作修訂了該工具，針對伴有或不伴有智力殘障的孤獨譜系成年人開發並測試，但也可用於一般人羣。

## 測試結構

- **14 道症狀嚴重程度題**，每題採用 5 點李克特量表
- **附加項目**（初始發生 2 題、自我反思 3 題、感知變化 1 題、參與支持 1 題）不計入總分
- **直接報告版本**（自評）和**照護者報告版本**（他評）兩種形式
- 核心 14 題約 4 分鐘，含前言及附加項目約 10 分鐘

### 症狀嚴重程度量表

每道題均以"至少在過去三個月裏"爲時間框架，要求受測者將當前體驗與自身平常情況進行比較。

**Likert 量表：**

| 分值 | 標籤 |
|------|------|
| 4 | 非常同意 (Strongly Agree) |
| 3 | 同意 (Agree) |
| 2 | 既不同意也不不同意 (Neither Agree nor Disagree) |
| 1 | 不同意 (Disagree) |
| 0 | 非常不同意 (Strongly Disagree) |

### 前言說明（量表填寫前的必要說明）

AASPIRE 強調，量表主體問題之前的說明信息**對準確作答至關重要**。其認知訪談研究發現，**部分使用者在不閱讀以下說明的情況下無法準確作答**。量表的使用者應逐條閱讀以下說明，再進入題目的作答。

#### "持續至少三個月"是什麼意思？

這些問題詢問的是已經持續至少三個月的經歷：

- 這些事情可能已經持續了 3 個月，也可能已經持續了多年
- 如果不記得確切持續了多久，也沒關係
- 不必每一天都經歷這些事情，也可以算持續三個月以上
- 但不要包括那些只持續了幾天或幾周的事情

#### 如何將經歷與"對你而言的常態"進行比較？

這些問題要求將經歷與對受測者而言的"常態"進行比較：

- "通常"指的是受測者認爲對自己而言正常或最典型的情況，這可能與當前的體驗相同，也可能不同
- 在這些方面的體驗有較大波動是正常的——思考當前的波動與通常的波動之間是否存在差異
- 如果長時間感覺不好，請回想開始感覺不好之前的情況

#### 回答示例：不同的人如何回答

以下是三個示例，說明不同情況的人應如何作答：

1. **Maria — 感官敏感性**
   - 響亮的聲音一直困擾着 Maria，但過去通常能夠應對
   - 現在情況糟糕多了，幾乎任何聲音都讓她感到非常痛苦
   - 她不記得更難忍受聲音具體是從什麼時候開始的，但認爲從大約一年前找到那份有壓力的新工作開始，情況一直在逐漸惡化
   - 對於"感官敏感性比平時更多"的題目，Maria 應選“**非常同意**”

2. **Sam — 情緒控制**
   - Sam 一直很難控制自己的情緒
   - Sam 控制情緒的能力沒有任何變化
   - 對於"比平時更難控制情緒"的題目，Sam 應選**非常不同意**”，因爲困擾程度沒有變化

3. **Jamal — 注意力**
   - Jamal 平時在集中注意力方面有些困擾
   - Jamal 剛剛收到了一些壞消息，他今天集中注意力比平時困難得多
   - 在收到壞消息之前，Jamal 集中注意力的情況和平時差不多
   - 對於"比平時更難集中注意力"的題目，Jamal 應選“**非常不同意**”，因爲這種變化還沒有持續至少三個月

### 附加項目

以下項目包含在成果項目關於孤獨譜系耗竭的調查部分中，**不計入總分**，僅供臨牀或研究參考。

**初始發生**（當症狀條目中至少一項選"同意"或"非常同意"時顯示）：

- **#15** 這些變化似乎是多久出現的？→ 數天/數週內迅速惡化 / 經過數月或數年的緩慢積累 / 其他
- **#16** 您認爲是什麼導致了這些變化？（多選）→ 身體疾病、手術或事故 / 重大生活事件或過渡期 / 期望與能力、資源和支持之間的長期不匹配 / 其他

**自我反思**：

- **#17** 在參與本次調查之前，您是否聽說過"孤獨譜系耗竭"？→ 是（瞭解）/ 是（不太瞭解）/ 否
- **#18** 您認爲您曾經經歷過孤獨譜系耗竭嗎？→ 是 / 否 / 我不知道
- **#19** 您認爲您現在正在經歷孤獨譜系耗竭嗎？→ 是 / 否 / 我不知道

**感知變化與參與支持**：

- 總體而言，在過去六個月中，本調查所詢問的事情是變得更糟、保持不變還是變得更好？
- 您在回答本部分調查問題時是否獲得了任何幫助？→ 是 / 否（如"是"，追問幫助類型）

## 評分

### 症狀嚴重程度

每道症狀題（題 1–14）採用 5 點李克特量表，從 0（非常不同意）到 4（非常同意），無反向計分。**總分爲題 1–14 之和**，範圍 0–56 分，分數越高表示耗竭越嚴重。附加項目（題 15–19 及感知變化、參與支持）不計入總分。

### 截斷值（暫定）

以下截斷值用於判斷臨牀顯著性孤獨譜系耗竭，應始終結合臨牀判斷使用：

| 總分範圍 | 臨牀顯著性可能性 |
|----------|--------------|
| 0–22 | 低可能性 |
| 23–32 | 中等可能性 |
| 33–56 | 高可能性 |

### 維度

初步分析支持該量表具有**單因子結構**（Nicolaidis et al., 2026a）。

| 維度 | 縮寫 | 題數 | 分數範圍 |
|------|------|------|----------|
| 孤獨譜系耗竭 | ABS_Total | 14 | 0–56 |

## 心理測量學特性

原始版 AASPIRE 孤獨譜系耗竭量表在 80 名孤獨譜系成年人中進行了試點測試（49% 女性、31% 男性、20% 其他性別；年齡 18–68 歲，均值 35.9 歲）。內部一致性 α=0.95。建構效度經先驗假設檢驗支持：與抑鬱（CESD-10）的相關 r=.56（p<.0001），與感知壓力（PSS-10）的相關 r=.68（p<.0001），當前自覺耗竭者的量表得分顯著高於非耗竭者（p<.0001）。ROC 曲線 AUC=0.85，顯示良好的分類能力。97% 的參與者認同研究團隊對孤獨譜系耗竭的定義。值得注意的是，掩飾行爲（CAT-Q）與耗竭得分無顯著相關（r=.04, p=.7）——研究團隊推測 CAT-Q 測量的是掩飾行爲的*執行*而非*壓力*，且處於耗竭狀態者可能更*難以*執行掩飾，因此橫斷面上無法捕捉二者的線性關係（[[RaymakerEtAl2026|Raymaker et al., 2026]]）。後續獨立樣本驗證亦報告了良好的心理測量學特性（[[BougoureEtAl2025|Bougoure et al., 2025]]; [[MantzalasEtAl2024|Mantzalas et al., 2024]]）。

修訂版正在作爲 AASPIRE Outcomes Study 的一部分進行心理測量學測試，該研究是一項縱向隊列研究，追蹤約 850 名孤獨譜系成年人的異質性樣本隨時間的變化（Nicolaidis et al., 2026a）。

| 指標 | 結果 |
|------|------|
| 內部一致性信度 | α = 0.95 |
| 2 周重測信度 | ICC = 0.92 |
| 內容效度 | 認知訪談數據支持 |
| 收斂效度與區分效度 | 強 |
| 6 個月變化反應度 | 良好 |
| 結構效度 | 單因子，初步分析支持 |

關於該量表結構和結構效度的額外結果尚待發表，將在可用時在 AASPIRE Measurement Toolkit 網站上分享。

## 照護者報告版本

照護者報告版本與直接報告版本採用相同的 14 道症狀條目結構，但措辭從第一人稱改爲第三人稱（"they"）。附加項目中的自我反思題被替換爲關於照護者信息來源的問題。

照護者報告版本同樣包含：
- 症狀嚴重程度（14 題，結構同自評版）
- 初始發生（#15–#16，判斷方式同自評版）
- 關於感知變化及知識來源的標準照護者報告項目
- "超鏈接"中的輔助說明內容

## 原始版與改編版的主要差異

| 維度 | 原始版 | 修訂版 (AABS-R14) |
|------|--------|-----------------|
| 條目數 | 27 題（症狀）+ 附加項目 | 14 題（症狀）+ 附加項目 |
| 時間框架 | 未明確指定 | 明確爲"至少在過去三個月裏" |
| 比較基準 | 未明確 | 明確要求與"自身平常情況"比較 |
| 可及性 | 原始措辭 | 經認知訪談優化，提高可理解性 |
| 情景描述 | 無 | 每條附具體示例 |
| 初始發生 | 無 | 新增 2 題 |
| 自我反思 | 無 | 新增 3 題 |
| 結構效度 | 多因子 | 單因子（初步） |

## 與神經多樣性的關係

[[孤獨譜系耗竭]]是[[神經多樣性範式]]中的核心概念——它不是個體內在的缺陷，而是**個體能力、資源、支持與環境期望之間長期不匹配的結果**。AABS-R14 的設計體現這一視角：

- 量表要求與"自身平常情況"比較，而非與神經典型常態比較——尊重個體的神經多樣性基線
- 附加項目 #16 將"期望與能力、資源和支持之間的長期不匹配"列爲耗竭的可能原因之一
- 採用 CBPR 方法開發，孤獨譜系社區成員全程參與
- 認知無障礙設計（情景描述、簡化措辭）體現了神經多樣性肯定實踐
