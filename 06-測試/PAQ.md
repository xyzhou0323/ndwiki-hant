# PAQ 珀斯述情障礙問卷

**PAQ**（Perth Alexithymia Questionnaire）是一份評估**述情障礙**（Alexithymia）的自填工具，由 Preece 等人於2018年開發。

> **本站測試**：[PAQ 珀斯述情障礙問卷](https://neuroxyz.cn/tests/paq.html)
> **中文版提示**：本測試目前僅有英文原版常模，**尚無中文版常模**。本站提供的百分位、閾值等解讀數據均基於英文原版，缺少本土化驗證。中文語境下的分數解讀存在文化差異風險，結果**僅供參考**，不構成任何臨牀判斷依據。

## 什麼是述情障礙

**述情障礙**指識別和描述自身情緒狀態的困難。它並非[[孤獨譜系障礙]]的專屬特徵——在一般人羣中也有分佈——但在孤獨譜系羣體中發生率顯著較高（約40-65%）。述情障礙與許多心理健康困難（尤其是焦慮和抑鬱）密切相關。

## 測試結構

- **24題**，約8分鐘完成
- 核心區分：**正性情緒 vs. 負性情緒**——述情障礙在這兩種情緒上的困難可能不對稱
- 三個子維度：情緒識別困難（DIF）、情緒描述困難（DDF）、外向性思維（EOT）
- 每題採用 **7點 Likert 量表**：

| 分值 | 標籤 |
|------|------|
| 1 | 非常不同意 |
| 2 | 不同意 |
| 3 | 有些不同意 |
| 4 | 中立 |
| 5 | 有些同意 |
| 6 | 同意 |
| 7 | 非常同意 |

### 題目-分量表對應

| 分量表 | 代碼 | 題目（1-indexed） | 含義 |
|--------|------|-------------------|------|
| 負性-情緒識別困難 | N-DIF | 2, 8, 14, 20 | 難以識別自己的負性情緒 |
| 正性-情緒識別困難 | P-DIF | 5, 11, 17, 23 | 難以識別自己的正性情緒 |
| 負性-情緒描述困難 | N-DDF | 1, 7, 13, 19 | 難以向他人描述自己的負性情緒 |
| 正性-情緒描述困難 | P-DDF | 4, 10, 16, 22 | 難以向他人描述自己的正性情緒 |
| 總體外向性思維 | G-EOT | 3, 6, 9, 12, 15, 18, 21, 24 | 較少關注內在情緒，偏向外部導向思維 |

### 複合分數公式

| 複合分      | 代碼    | 計算公式                                              |
| -------- | ----- | ------------------------------------------------- |
| 總體情緒識別困難 | G-DIF | = N-DIF + P-DIF                                   |
| 總體情緒描述困難 | G-DDF | = N-DDF + P-DDF                                   |
| 負性情緒評估困難 | N-DAF | = N-DIF + N-DDF                                   |
| 正性情緒評估困難 | P-DAF | = P-DIF + P-DDF                                   |
| 總體情緒評估困難 | G-DAF | = G-DIF + G-DDF (= N-DIF + P-DIF + N-DDF + P-DDF) |
| 述情障礙總分   | TOTAL | = 全部 24 題得分之和                                     |

## 結果解讀

### 整體分級

基於總分的分級（參照常模均值 82，SD=31）：

| 總分 | 分級 |
|------|------|
| < 51 | 無述情障礙（低於均值-1SD） |
| 51 - 112 | 平均水平（均值±1SD 範圍內） |
| >= 113 | 述情障礙（高於均值+1SD） |

> **注意**：述情障礙可能是一個連續譜系（低 -> 中 -> 高），而非截然不同的分類。以上分級僅爲便於理解的參考框架。

### 羣體特徵提示

- 普通人羣中，**負性情緒維度**（N-DIF、N-DDF）通常得分**高於**正性情緒維度（P-DIF、P-DDF）
- 述情障礙人羣的負性情緒困難通常更爲突出

## 在神經多樣性範式中的位置

述情障礙是[[神經多樣性範式]]與[[常態範式]]之間緊張關係的典型案例：

- 述情障礙確實造成個體的主觀困擾——消除它的努力有合理的基礎
- 但分類須謹慎：情緒覺察方面的差異**不必然**是述情障礙——某些差異反映的可能是不同的情緒加工風格，而非能力缺陷
- [[神經多樣性範式]]要求我們區分"造成困擾"和"僅偏離規範"——前者可以干預，後者不應被病理化

## 與其他概念的關係

述情障礙與內感覺密切相關——對身體信號的覺察困難可能導致情緒覺察困難（[[MAIA-2]]評估內感覺）。述情障礙也與同理心有關——識別自身情緒的困難可能延伸到理解他人情緒（[[PES]]評估同理心）。
