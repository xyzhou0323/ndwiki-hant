# 神經多樣性 Wiki（繁體中文版）

本倉庫是[神經多樣性 Wiki](https://github.com/xyzhou0323/ndwiki-cn) 的繁體中文（zh-hant）版本，採用 ISO 15924 腳本代碼 `zh-hant`，涵蓋臺灣、香港、澳門等所有繁體使用者，不預設地區。

> **⚠️ 術語轉換目前由 AI（OpenCC）自動完成，全部頁面均待人工討論與審校。** 字形轉換（簡→繁）由機器處理，但詞彙層面的選擇——例如確定採用「自閉光譜」——尚未經過社群討論，所有術語均為暫定版本，隨時可能調整。

**線上閱讀**：[neuroxyz.cn/wiki-hant](https://neuroxyz.cn/wiki-hant/)

## 快速開始

用 [Obsidian](https://obsidian.md) 打開本倉庫即可瀏覽，或直接查看 GitHub 上的 Markdown 檔案。

入口頁面：
- **新讀者** → [[閱讀路線]]：四層遞進導航，四個起點
- **查概念** → [[00-知識地圖|知識地圖]]：八部分完整索引

## 與簡體版的關係

| | 簡體版 | 繁體版 |
|---|---|---|
| 路徑 | `wiki/` | `wiki-zh-hant/` |
| 倉庫 | [ndwiki-cn](https://github.com/xyzhou0323/ndwiki-cn) | 本倉庫 |
| 腳本 | zh-hans | zh-hant |
| 內容來源 | 作者原創（zh-hans） | OpenCC 轉換 + 人工審校 |

簡體版為主倉庫。簡體頁面更新後，維護者執行 `sync-zh-hant.py` 將變更自動轉為繁體並推送至本倉庫。繁體協作者審校自動轉換的內容，修正地區術語差異。

## 協作

歡迎繁體中文使用者參與審校和共建！詳細流程見 [COLLABORATING.md](COLLABORATING.md)。

### 簡要流程

1. 在本倉庫中搜尋 frontmatter 包含 `needs-review: true` 的頁面
2. 審校術語和轉換質量，修正後移除 `needs-review: true`
3. 提交 PR 到 `main` 分支

### 地區術語

> **以下對照表為 AI 暫定版本，未經社群討論確認。** 簡繁轉換（OpenCC s2t）處理的是字形層面，不處理詞彙層面的地區差異。例如，「孤獨譜系」經 OpenCC 轉為「孤獨譜系」，但臺灣／香港社群可能偏好完全不同的譯法（如確定採用「自閉光譜」）。所有術語均需人工審校與協商。

以下為常見差異範例，**僅供參考，非最終決定**：

| 簡體（zh-hans） | 臺灣（zh-hant） | 香港（zh-hant） |
|---|---|---|
| 孤独谱系 | 自閉光譜 | 自閉光譜 |
| 注意缺陷多动障碍 | 注意力不足過動症 | 專注力失調/過度活躍症 |
| 生活质量 | 生活品質 | 生活質素 |
| 残障 | 身心障礙 | 殘疾 |

審校時如遇到地區差異，建議在正文中以括號加註（如「孤獨譜系（臺：自閉光譜）」，或在[[術語翻譯對照表]]中記錄。頁面主體採用臺灣用語為預設，香港用語以括號加註。

## 維護

- **同步**：簡體版維護者透過 `sync-zh-hant.py` 將變更推送至本倉庫

- **連結**：每個繁體頁面 frontmatter 包含 `zh-hans` 欄位指向對應簡體頁面
