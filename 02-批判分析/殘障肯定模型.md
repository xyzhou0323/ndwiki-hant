# 殘障肯定模型

殘障肯定模型（Affirmative Model of Disability）由 John Swain 和 Sally French 於2000年提出。它拒絕將殘障框定爲"個人悲劇"的主導敘事，轉而主張將殘障理解爲一種**積極身份**和集體文化認同的來源。

## 起源

Swain 和 French 在2000年的論文 *Towards an Affirmation Model of Disability* 中首次系統闡述了這一模式。兩人的觀察基礎是：殘障權利活動家和殘障者社羣在日常實踐中早已踐行"肯定"態度——拒絕憐憫、拒絕"勵志"敘事、拒絕被定義爲缺失——但學術理論尚未爲這一立場提供明確的概念表述。

這一模式並非替代社會模型，而是社會模型的**補充和延伸**——社會模型關注的是殘障在社會結構中的產生，肯定模型關注的是殘障者在身份和文化層面的自我定義。

## 核心主張

### 拒絕個人悲劇敘事

主導文化將殘障敘事爲一連串個人悲劇：

- 殘障 = 失去"完整"的人生
- 殘障者 = 憐憫對象
- 殘障者取得任何成就 = "勵志故事"（被視爲例外）

殘障權利倡導者一貫拒絕這一框架。肯定模型將此拒絕理論化：殘障不是發生在個體身上的悲劇，而是人類多樣性的有效形式。

### 積極身份

肯定模型主張殘障可以被體驗爲積��的身份維度：

- **同一性**（identity）：殘障身份是"我是誰"的構成性部分，不是被強加的外部標籤
- **自尊**（pride）：拒絕羞恥，將殘障視爲值得自尊的集體身份
- **文化**（culture）：殘障者發展出共享的語言、幽默、藝術和政治傳統——一種活的集體文化
- **社羣**（community）：殘障者共享壓迫經驗以及在非包容世界中生活的實踐知識

### 社會模型中的"肯定"根基

肯定模型建立在社會模型已有的核心區分之上。社會模型指出殘障由社會製造；肯定模型進一步指出：既然殘障是社會建構的，那麼對殘障的否定和貶低同樣是被建構的價值判斷——可以被拒絕和替代。

## 與神經多樣性運動的關係

肯定模型與神經多樣性運動在多個層面產生共鳴：

### 神經殊異自尊

神經多樣性運動中出現了與肯定模型平行的概念和實踐：

- **孤獨譜系自尊**（Autistic Pride）：每年6月18日的"孤獨譜系自尊日"由 Aspies for Freedom 在2005年創建。拒絕"神經典型至上"，堅持孤獨譜系是一種應該被肯定而非被消滅的存在方式
- **瘋狂自尊**（Mad Pride）：精神健康服務使用者/倖存者運動中的肯定身份政治——拒絕"病患"身份，擁抱"瘋狂"作爲一種體驗和身份
- **神經彩虹**（NeuroRainbow）符號：將"無限"符號與彩虹顏色結合，象徵神經多樣性的價值——與 LGBT 自尊旗的意義平行
- **孤獨譜系不是悲劇**（Autism Is Not a Tragedy）、**#ActuallyAutistic** 等線上社羣實踐

### 肯定式語言

肯定模型推動的語言變革與神經多樣性運動高度平行：
- 醫學語言：person with autism（患有孤獨症的人） → 肯定語言：autistic person（孤獨譜系者）
- 醫學語言：disorder/deficit（病症/缺陷） → 肯定語言：difference/variation（差異/變體）
- 拒絕功能標籤——沒有"高功能"和"低功能"，只有不同的支持需求

### 與神經多樣性範式的深層一致

[[神經多樣性範式]]的第一條原則——"神經多樣性是一種應當被肯定的自然人類變異"——與肯定模型的核心主張在結構上完全一致：

| 肯定模型 | 神經多樣性範式 |
|----------|---------------|
| 拒絕殘障 = 個人悲劇 | 拒絕神經殊異性 = 病理 |
| 殘障作爲積極身份 | 神經殊異性作爲被肯定的差異 |
| 集體文化認同 | 神經少數羣體政治認同 |
| 拒絕憐憫和勵志敘事 | 拒絕"可治癒"的醫學敘事 |

兩者分別從殘障研究和神經多樣性運動兩條獨立的進路得出了相同的結論：**人類差異不需要被消除，它們需要被肯定**。

## 批評與實用考量

肯定模型面臨的主要批評與實用考量：

- **與政治現實的張力**：當殘障者被國家縮減支持、剝奪福利、乃至拒絕生存權時，"肯定"的修辭可能爲緊縮政策提供道德遮羞布（參見 [[JonesOrchard2024|Jones & Orchard 2024]] 對神經多樣性撇脂效應的批判）
- **損傷效應的承認**（參見 [[殘障的社會模型#損傷效應]]）可能在不否定肯定立場的前提下保持對殘障痛苦維度的有效回應
- **並非所有人都會將殘障體驗爲積極身份**——肯定模型應作爲可選擇的有尊嚴敘事被提供，而非成爲新的義務
